# تشغيل النظام محليًا (داخل المكتب فقط)

هذا المشروع الآن يحتوي **نسخة تشغيلية أولية (MVP Lite)** يمكنك تشغيلها فورًا على الشبكة المحلية.

## 1) التشغيل السريع

## المتطلبات
- Python 3.11+
- جهاز الخادم داخل LAN

## خطوات التشغيل
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

بعد التشغيل افتح من نفس الجهاز:
- `http://127.0.0.1:8000`

أو من أي جهاز داخل الشبكة:
- `http://<LOCAL_SERVER_IP>:8000`

مثال:
- `http://192.168.1.20:8000`

## بيانات الدخول الافتراضية
- المستخدم: `owner`
- كلمة المرور: `admin123`

> غيّر كلمة المرور مباشرة بعد أول تشغيل في النسخة الإنتاجية.

---

## 2) ماذا يفعل الإصدار الحالي؟
- تسجيل دخول بجلسة محلية.
- إنشاء مستند رسمي بسيط.
- توليد 3 صيغ:
  - DOCX
  - PDF
  - XLSX
- حفظ الملفات داخل:
  - `app/storage/generated/`
- حفظ البيانات الوصفية داخل SQLite:
  - `app/storage/app.db`

---

## 3) ملاحظات LAN والأمان
- التطبيق لا يحتاج إنترنت للتشغيل.
- اجعل الوصول محصورًا من داخل LAN عبر الجدار الناري (Firewall).
- لا تفتح المنفذ 8000 للإنترنت العام.

---

## 4) استكشاف الأخطاء السريع
- إذا ظهر خطأ مكاتب PDF/DOCX/XLSX:
  - تأكد أن `pip install -r requirements.txt` اكتمل.
- إذا المنفذ مستخدم:
  - شغّل على منفذ آخر:
  ```bash
  uvicorn app.main:app --host 0.0.0.0 --port 8010
  ```
- إذا لا يمكن فتحه من أجهزة الشبكة:
  - تأكد من IP الصحيح وجدار الحماية على جهاز الخادم.
